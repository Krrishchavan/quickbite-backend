# QuickBite - Backend API

Food delivery application backend built with Node.js, Express, and MongoDB.

## Features
- JWT Authentication
- Role-based access (Customer, Restaurant, Rider)
- Order management
- Real-time delivery tracking
- Auto rider assignment

## Tech Stack
- Node.js + Express
- MongoDB + Mongoose
- JWT for authentication
- Vercel serverless deployment

## Environment Variables
Required for deployment:
```
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
NODE_ENV=production
```

## Deployment
This backend is configured for Vercel serverless deployment. See main project README for full deployment instructions.
